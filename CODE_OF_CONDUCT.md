# Code of Conduct

## Keep It Simple

1. **Don't be an asshole**
2. **Help others when you can**
3. **Assume good intent**

That's it. That's the code of conduct.

---

## Expanded (If You Need It)

### Be Chill

- Disagreements are fine. Attacks aren't.
- Critique code, not people.
- If someone's new, help them. You were new once.

### Respect the Vibe

## Enforcement

If someone's being a problem:

1. First time: Friendly reminder
2. Second time: Timeout
3. Third time: Gone

We're not building a bureaucracy. Bad actors get removed. Good actors get celebrated.

---

## Questions?

If something feels off, flag it. Open an issue or reach out directly to maintainers.
